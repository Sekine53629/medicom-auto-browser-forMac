【SETUP_GUIDE.md を Word形式にする方法】

===============================================
方法1: Pandoc を使う（推奨・自動変換）
===============================================

1. Pandoc をインストール
   https://pandoc.org/installing.html
   → Windows用インストーラーをダウンロード＆インストール

2. 「convert_to_word.bat」をダブルクリック
   → 自動的に SETUP_GUIDE.docx が作成されます

===============================================
方法2: Microsoft Word で開く（手動・簡単）
===============================================

1. Microsoft Word を開く

2. 「ファイル」→「開く」

3. ファイルの種類を「すべてのファイル (*.*)」に変更

4. SETUP_GUIDE.md を選択して開く

5. 「ファイル」→「名前を付けて保存」

6. ファイルの種類を「Word 文書 (*.docx)」に変更

7. 保存

===============================================
方法3: オンライン変換サービス（インストール不要）
===============================================

以下のサイトにアクセス：
https://cloudconvert.com/md-to-docx

1. 「Select File」をクリック

2. SETUP_GUIDE.md を選択

3. 「Convert」をクリック

4. 変換完了後、ダウンロード

===============================================
方法4: Visual Studio Code を使う
===============================================

1. VS Code で SETUP_GUIDE.md を開く

2. 拡張機能「Markdown PDF」をインストール

3. Ctrl+Shift+P を押して「Markdown PDF: Export (docx)」を実行

===============================================

おすすめ: 方法2（Microsoft Word）が最も簡単です！
